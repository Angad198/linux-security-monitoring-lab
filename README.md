# Linux Security Monitoring & Log Analysis Lab

## 📌 Overview
This project demonstrates Linux-based security log monitoring and analysis. It simulates a basic SOC (Security Operations Center) environment where system logs are reviewed to detect authentication events, privileged access, and potential security issues.

## 🛠️ Tools & Technologies
- Ubuntu Linux (WSL2)
- Bash Shell
- grep, wc commands
- System authentication logs (/var/log/auth.log)
- Git & GitHub

## 🔍 Features
- Collected and analyzed system authentication logs
- Identified sudo (privileged) activity
- Tracked login sessions and system events
- Generated a security analysis report
- Organized logs, docs, and analysis files

## 📊 Key Skills Demonstrated
- Linux log analysis
- Security event monitoring
- System administration basics
- Incident reporting
- Command-line data filtering

## 📁 Project Structure
- logs/ → system log samples
- docs/ → reports and screenshots
- analysis/ → security findings
- scripts/ → automation scripts (if any)

## 🚀 Outcome
This project demonstrates practical skills in Linux system administration and cybersecurity monitoring, relevant for IT support and SOC analyst roles.
